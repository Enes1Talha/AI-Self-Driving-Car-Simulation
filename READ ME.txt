# 🚗 AI Self-Driving Car Simulation with NEAT

This project demonstrates the evolution of a self-driving car using the **NEAT (NeuroEvolution of Augmenting Topologies)** algorithm. Starting from complete chaos, the AI learns to navigate a track, avoid obstacles, and optimize its speed through natural selection.


## 🧬 How It Works?
The simulation uses a **Genetic Algorithm** process:
1.  **Generation 0:** 100 cars start with random neural networks.
2.  **Selection:** Cars that crash or stay idle are eliminated.
3.  **Evolution:** The "fittest" cars (those that travel the furthest) breed to create the next generation.
4.  **Mutation:** Random changes are introduced to discover better driving strategies.

## 🛠️ Tech Stack
* **Python 3.x**
* **Pygame** (For simulation and rendering)
* **NEAT-Python** (For the evolutionary neural network)

## 🚀 Installation & Run

1.  Clone the repository:
    ```bash
    git clone [https://github.com/Enes1Talha/AI-Self-Driving-Car.git](https://github.com/Enes1Talha/AI-Self-Driving-Car.git)
    cd AI-Self-Driving-Car
    ```

2.  Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

3.  Run the simulation:
    ```bash
    python main.py
    ```

## 🧠 Evolution Highlights
* **Gen 0-2:** Complete chaos, mostly crashing.
* **Gen 3:** First car completes the lap.
* **Gen 30+:** "Swarm Intelligence" achieved. Cars drive perfectly in sync.
